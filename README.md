# KTextAddons

Various text handling addons:

- Auto-correction API
- Text to Speak widgets
- Text translator widgets (Several backend)
- Text Emoticon support
- LLM support widgets/backend (ollama)

## Dependencies

- extra-cmake-modules

Qt:

- Qt6Core
- Qt6Network
- Qt6Widgets
- Qt6Keychain
- Qt6UiPlugin

KF6:

- KF6Archive
- KF6CoreAddons
- KF6I18n
- KF6Sonnet
- KF6KIO
- KF6SyntaxHighlighting
- KF6Config
- KF6ColorScheme
- KF6TextWidgets
